# 퍼즐 게임 PRD

## 목적

간단한 그림 짝맞추기 게임을 만든다. 사용자는 같은 그림 2개를 찾으면 해당 카드가 맞춰지고, 모든 카드를 맞추면 클리어된다.

## 참고 UI

- 같은 경로의 `ui.png`

## 핵심 요구사항

- 4x4 카드 보드
- 같은 그림 2개를 찾아서 맞추는 게임
- 시도 횟수 표시
- 다시 시작 버튼 제공
- 모든 짝을 맞추면 완료 메시지 표시

## 화면 구성

- 상단 제목
- 설명 문구
- 좌측에 시도 횟수
- 우측에 다시 시작 버튼
- 가운데 4x4 카드 그리드
- 하단 완료 메시지

## 동작 규칙

- 게임 시작 시 카드 16장을 섞는다.
- 카드를 클릭하면 그림이 드러난다.
- 서로 같은 그림이면 맞춘 카드로 고정한다.
- 서로 다르면 잠시 뒤 다시 숨긴다.
- 두 장을 맞출 때마다 시도 횟수를 1 증가시킨다.
- 모든 카드가 맞춰지면 축하 메시지를 보여준다.
- 다시 시작 버튼을 누르면 처음 상태로 돌아간다.
