# 퍼즐 게임 PRD

## 목적

틀린 그림 찾기 게임이고, 단계별로 난이도가 상승하고, 각각의 제한 시간이 있으며, 시작/종료 화면을 넣은 단계형 게임을 만든다.

## 게임 개요

- 사용자는 시작 화면에서 게임을 시작한다.
- 게임은 1단계부터 3단계까지 난이도 순서로 진행된다.
- 각 단계마다 제한 시간 안에 틀린 그림을 맞춰야 한다.
- 모든 단계를 완료하면 종료 화면과 축하 메시지를 보여준다.

## 화면 구성

- 시작 화면
  - 게임 제목
  - 시작 버튼
- 게임 화면
  - 맞춰야 되는 틀린 그림 갯수/맞춘 틀린 그림 갯수
  - 남은 시간
  - 그림 바꾸기 버튼
  - 좌/우 틀린 그림 배치
  - 안내 메시지
- 종료 화면
  - 축하 메시지
  - 다시 하기 버튼

## 단계 구성

- 1단계: 틀린 그림 4개 맞추기
- 2단계: 틀린 그림 6개 맞추기
- 3단계: 틀린 그림 8개 맞추기

## 동작 규칙

- 사용자가 시작 버튼을 누르면 1 단계부터 게임을 시작한다.
- 각 단계는 좌/우로 그림을 배치하여 구성한다.
- 그림에는 왼쪽이 원본 그림, 오른쪽이 틀린 그림을 맞춰야 되는 그림으로 배치한다.
- 사용자가 그림을 선택할 경우 어떤 부분을 클릭했는지 알 수 있도록 한다.
- 왼쪽/오른쪽의 틀린 그림을 맞췄을 경우 해당 부분을 동그라미로 표시한다.
- 단계별 제한 시간이 0이 되면 게임오버 처리한다.
- 한 단계를 모두 맞추면 다음 단계로 자동 이동한다.
- 마지막 단계까지 완료하면 종료 화면을 보여준다.
- 다시 시작 버튼은 현재 진행 중인 게임을 다시 시작한다.
