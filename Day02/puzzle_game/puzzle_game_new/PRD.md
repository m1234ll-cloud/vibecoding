# 퍼즐 게임 PRD

## 목적

그림 짝맞추기 퍼즐에 레벨 진행, 제한 시간, 시작/종료 화면을 넣은 단계형 게임을 만든다.

## 게임 개요

- 사용자는 시작 화면에서 게임을 시작한다.
- 게임은 쉬움부터 시작해 보통1, 보통2, 어려움 순서로 진행된다.
- 각 단계마다 제한 시간 안에 모든 짝을 맞춰야 한다.
- 모든 단계를 완료하면 종료 화면과 축하 메시지를 보여준다.

## 목표
- 실제 사용시스템처럼 폴더 구성도 하고, HTML, CSS, JavaScript를 독립적으로 파일로 만든다.

## 화면 구성

- 시작 화면
  - 게임 제목
  - 시작 버튼
- 게임 화면
  - 시도 횟수
  - 점수
  - 남은 시간
  - 다시 시작 버튼
  - 퍼즐 보드
  - 안내 메시지
- 종료 화면
  - 축하 메시지
  - 최종 결과 요약
  - 다시 하기 버튼

## 단계 구성

- 쉬움: 3x3
- 보통1: 4x4
- 보통2: 5x5
- 어려움: 6x6

## 동작 규칙

- 사용자가 시작 버튼을 누르면 쉬움 단계부터 게임을 시작한다.
- 각 단계는 정사각형 보드로 구성한다.
- 보드에는 같은 그림 2장이 한 쌍이 되도록 배치한다.
- 3x3, 5x5처럼 홀수 칸이 생기는 단계는 빈 칸 1개를 둔다.
- 카드를 클릭하면 카드가 뒤집히고 그림이 보인다.
- 같은 그림 2장을 맞추면 해당 카드는 고정하고 점수를 10점 올린다.
- 두 장이 다르면 잠시 보여준 뒤 다시 뒤집는다.
- 두 장을 선택할 때마다 시도 횟수를 1 증가시킨다.
- 단계별 제한 시간이 0이 되면 게임오버 처리한다.
- 한 단계를 모두 맞추면 다음 단계로 자동 이동한다.
- 마지막 단계까지 완료하면 종료 화면을 보여준다.
- 다시 시작 버튼은 현재 진행 중인 게임을 다시 시작한다.

## 점수와 시간

- 매칭 성공 시 10점씩 증가한다.
- 점수는 단계가 넘어가도 누적된다.
- 제한 시간은 단계별로 다르게 설정한다.
  - 쉬움: 45초
  - 보통1: 75초
  - 보통2: 100초
  - 어려움: 140초

## 상태 전환

- 시작 화면 -> 게임 화면: 시작 버튼 클릭
- 게임 화면 -> 다음 단계: 현재 단계 클리어
- 게임 화면 -> 종료 화면: 마지막 단계 클리어
- 게임 화면 -> 시작 화면: 다시 하기
- 게임 화면 -> 게임오버: 제한 시간 종료

## 후속 확장 여지

- 난이도 선택 콤보박스는 현재 화면에서 숨겨 둔 상태다.
- 필요 시 시작 단계 선택 기능으로 다시 노출할 수 있다.
